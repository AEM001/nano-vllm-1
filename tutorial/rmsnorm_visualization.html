Understanding Root Mean Square Layer Normalization

❓
What is RMSNorm?

RMSNorm (Root Mean Square Normalization) is a simplified version of Layer Normalization used in modern LLMs like Llama and Qwen. Instead of normalizing by both mean and variance, RMSNorm only uses the root mean square of the values, making it faster and more stable.

📐
The Magic Formula
RMSNorm(x) = x / √(mean(x²) + ε) × γ
Where:
• x = input vector
• ε (epsilon) = small number to prevent division by zero (default: 1e-6)
• γ (gamma) = learnable scale parameter (initialized to 1)
• mean(x²) = average of squared values = RMS (Root Mean Square)

⚖️
RMSNorm vs LayerNorm
🔶 LayerNorm

Formula: (x - mean) / √(var + ε) × γ + β

Properties:

  • Subtracts mean
  • Divides by std dev
  • Has learnable scale (γ) and bias (β)
  • More parameters
🔷 RMSNorm

Formula: x / √(mean(x²) + ε) × γ

Properties:

  • No mean subtraction
  • Uses RMS instead of std dev
  • Only learnable scale (γ)
  • ~30% faster, fewer parameters
🔗
Residual Connection (add_rms_forward)

Modern transformers use residual connections to help gradients flow better during training. The add_rms_forward method combines the residual connection with normalization in one efficient operation.
